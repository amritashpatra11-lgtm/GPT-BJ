# Phone-only deployment

## GitHub Pages

1. Open GitHub in Chrome and enable **Desktop site** if necessary.
2. Create a new public repository named `amritesh-cosmic-site`.
3. Tap **Add file → Upload files**.
4. Upload `index.html`, `styles.css`, `app.js`, `README.md`, and `LICENSE.txt`.
5. Create an `assets` folder and upload `favicon.svg` inside it.
6. Commit the files.
7. Open **Settings → Pages**.
8. Under **Build and deployment**, select **Deploy from a branch**.
9. Choose `main`, choose `/ (root)`, and save.
10. Wait for GitHub to show the public site address.

The site needs no npm installation, API key, database, or paid hosting.
