# AMRITESH Cosmic Website

A zero-cost, responsive, animated landing page reconstructed from the supplied reference video.

## Included

- Animated orbital particle hero
- Particle funnel and performance metric
- Flowing data landscape
- Animated black-hole scene
- Interactive documentation section
- Responsive mobile navigation
- Smooth reveals, counters, and pointer interaction
- Creator branding: **AMRITESH**
- No paid assets, API keys, frameworks, or build process

## Open locally

Open `index.html` in a browser. For the most reliable local preview, run a small web server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish free with GitHub Pages

1. Create a public GitHub repository.
2. Upload all files from this folder, preserving the `assets` directory.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.

## Publish free with Cloudflare Pages

1. Import the GitHub repository into Cloudflare Pages.
2. Framework preset: `None`.
3. Build command: leave blank.
4. Output directory: `/`.

## Customise

- Main text: edit `index.html`.
- Colours, spacing, and layout: edit `styles.css`.
- Canvas scenes and motion: edit `app.js`.
- Contact button: replace `hello@example.com` in `index.html`.

## Technology

Plain HTML, CSS, JavaScript, and Canvas 2D. The site is entirely self-contained and uses no external libraries.
